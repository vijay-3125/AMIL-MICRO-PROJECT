# AMIL-MICRO-PROJECT
This project focuses on Waste Classification using Artificial Neural Networks (ANN). The model is trained to classify different types of waste images into categories using deep learning techniques. This helps in automating waste segregation, which is useful for smart cities and environmental sustainability.  
Waste Classification using Artificial Neural Network (ANN)
Project Overview

This project focuses on waste classification using Artificial Neural Networks (ANN). The model is trained to classify different types of waste images automatically using deep learning techniques. This system helps improve waste segregation and recycling processes and contributes to environmental sustainability.
The dataset is obtained from Kaggle and processed using TensorFlow and Keras for training and evaluation.

Objectives:
Automatically classify waste images
Build an ANN model using TensorFlow and Keras
Train and validate the model
Evaluate model performance
Visualize model accuracy and confusion matrix

Technologies Used:
Python
TensorFlow
Keras
NumPy
Matplotlib
Scikit-learn
Kaggle Dataset

Dataset:
The dataset used in this project is a waste classification dataset obtained from Kaggle. It contains images of different types of waste categorized into multiple classes. The dataset is divided into training and validation sets for model development and evaluation.
Project Workflow
Data Collection
The dataset is downloaded from Kaggle and organized into different categories of waste images.

Data Preprocessing:
Image resizing
Data normalization
Splitting dataset into training and validation sets

Model Architecture:
The Artificial Neural Network consists of:
Input layer
Flatten layer
Hidden dense layers with ReLU activation
Output layer with Softmax activation

Model Training:
The model is trained using the training dataset and validated using the validation dataset. The training process includes multiple epochs to improve model accuracy.

Evaluation Metrics:
Accuracy
Classification Report
Confusion Matrix

Output:
Model accuracy visualization
Confusion matrix
Classification report
Trained model file

Applications:
Smart waste management
Recycling automation
Environmental monitoring
Smart city applications
